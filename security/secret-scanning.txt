Secret scanning (free)

Recommended tools:
- Gitleaks (fast, simple)
- TruffleHog (strong detectors)

This repo includes:
- GitHub Actions workflow: .github/workflows/secret-scan-gitleaks.yml
- GitLab CI template: ci-templates/gitlab-ci/secret-scan.gitlab-ci.yml

Tip:
- Treat any secret detection as a stop-the-line event.
- Rotate secrets immediately if exposed.
